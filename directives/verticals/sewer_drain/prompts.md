# Sewer & Drain — Agent Prompt Language

## Invoice agent prompt additions
- A rural Maine septic customer should feel like they got a fair deal from a neighbor
- Use plain trade language: "Septic pump-out — 1,000 gal. tank", "Outlet baffle replacement"
- Bad: "I pumped carol duggens 1000 gallon tank" → Good: "1,000 Gallon Septic Tank Pumping"
- Default payment: check, cash, or Venmo

## Proposal agent prompt additions
- If job type is unclear, default to inspection + pump
- If customer mentions it has been over 3 years, recommend pump
- Never quote $0 for any line item

## Scheduling notes
- Jobs typically 1-3 hours
- Emergency same-day, standard 3-7 day booking window

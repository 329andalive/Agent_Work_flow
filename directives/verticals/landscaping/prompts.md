# Landscaping — Agent Prompt Language

## Invoice agent prompt additions
- Customers expect clear line items: what was done, how long, what materials
- Use plain trade language: "Lawn mowing — front and back", "Spring cleanup — debris removal"
- Labor and materials should be separate line items when both apply
- Never round material quantities — if 3.5 yards of mulch, say 3.5 yards

## Proposal agent prompt additions
- Always ask or estimate the lawn size — price depends on it
- If scope is unclear, default to a basic mow + edge + cleanup estimate
- Snow removal: always clarify if per-visit or seasonal contract
- Never quote $0 for any line item

## Scheduling notes
- Mowing jobs typically 1-3 hours depending on size
- Cleanup jobs 2-6 hours
- Snow removal: same-day or next morning
- Mulch and install jobs: book 3-7 days out
